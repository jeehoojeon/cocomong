# 코코몽 에코파크 React 프로젝트

이 프로젝트는 코코몽 에코파크 웹사이트를 React로 구현한 것입니다.

## 설치 방법

1. 프로젝트 클론
```bash
git clone [repository-url]
cd cocomong-react
```

2. 의존성 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm start
```

## 사용된 기술

- React 18
- Swiper.js
- Lucide React Icons
- CSS3

## 프로젝트 구조

```
src/

  │   ├── ScrollBar.jsx
  │   ├── Section1.jsx

  │   ├── Section3.jsx
  │   ├── Section4.jsx
  │   └── Section5.jsx

```
